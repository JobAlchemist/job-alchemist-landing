# Recommendation Rubric

Use this rubric in Mode 1 (recommend pages).

## Prioritisation criteria
Score each candidate page (1-5):
1. Intent demand: Is the question/search intent common and clear?
2. Audience fit: Does it serve students, recent graduates, professionals, or HR?
3. Gap fit: Is this missing or weak in current site coverage?
4. Conversion adjacency: Can it naturally link to product pages without forced CTA language?
5. AEO suitability: Can the page answer the question directly in structured, quotable sections?

## Candidate page types
1. How-to tutorials (process and checklists)
2. Comparisons (tools/workflows)
3. Glossary pages (term clusters with practical usage)
4. Audience-specific guides (students/recent grads/professionals)
5. Scenario pages (career switch, interview prep, fit decisions)

## Recommendation output format
For each proposed page include:
1. Page title
2. Primary intent/query
3. Target audience
4. Value/why now
5. Internal links to include
6. Expected external sources to cite
7. Priority score

## Confirmation step
Always ask user to confirm which pages to proceed with before creating content.
